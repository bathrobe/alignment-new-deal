Visual Novel Kit

Skills & workflow for interactive fiction with Claude Code

1. Create
/story-architect NEW
Brainstorm premise, arcs, axes, characters. Outputs arc.md with scene checklist.
↓
/world-builder NEW
Setting, technology, mood, tone, rules. Outputs world.md bible.
→
2. Style
/style-discover NEW
Generate test images in different aesthetics. Pick one, build style lock + reference suite.
↓
/ui-designer NEW
Design sidebar, buttons (4 states), fonts, results screen. Output HTML mockup.
↓
/background-generator
Style lock prefix/suffix + auto-attached refs. Generates scene backgrounds.
Style References
4 mood refs (day exterior, interior, outdoor hope, night). Auto-attached to every generation.
→
3. Build
/scene-author
Write Ink dialogue + generate background per scene. Iterative with stop-and-review at each step.
↓
/run-all
Execute remaining checklist tasks sequentially with fresh context per task.
