Design Language

Core idea

Everything is flat color + bold black outline + hard shadow. Like a comic panel or screenprint. No gradients, no bevels, no glass effects. Depth comes from the offset shadow only.

Shadow system

/* Resting */
box-shadow: 4px 4px 0 #2B2D2F;

/* Hover (lifts) */
transform: translate(-1px, -1px);
box-shadow: 5px 5px 0 #2B2D2F;

/* Active (presses into shadow) */
transform: translate(2px, 2px);
box-shadow: 1px 1px 0 #2B2D2F;

transition: all 80ms ease;

What's different from merge-lane

✕ No gradients
✕ No beveled borders
✕ No inset shadows
✕ No hardware rivets
✓ Flat fills + bold outlines
✓ Hard offset shadows (comic drop shadow)
✓ Halftone dot texture
✓ Choices fill on hover (not slide)
